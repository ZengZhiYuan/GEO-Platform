## 执行规则摘要（§2–§4，行 28–175）

- **范围：** 仅 `backend` 接口缺口补齐；暂不开发/验收 `frontend`。
- **事实来源：** `docs/原型功能_API映射整合精简版.md`（唯一口径）；旧映射文档仅作历史参考。
- **API 前缀：** 主前缀 `/api/geo-monitoring`；兼容 `/api/v1/geo-monitoring`。
- **环境：** 后端命令必须使用 `backend/.venv`；文本输出默认 UTF-8（见 `AGENTS.md`）。
- **测试先行：** 先写失败测试，再最小实现；单 Task 验收见各 Task「验收命令」。
- **文档同步：** 每完成一 Task，更新 `docs/API接口文档.md`、`docs/API测试文档.md`，并在 `docs/原型功能_API映射整合精简版.md` 标注已覆盖缺口（§8）。
- **Superpowers（代码 Task 必用）：** 见 `.cursor/rules/superpowers-dev-workflow.mdc`。
- **CodeGraph（有源码改动且验收通过后）：**
  ```powershell
  codegraph status
  codegraph sync
  codegraph status
  ```
